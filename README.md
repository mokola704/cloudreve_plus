# CloudreveV4_Plus
基于CloudreveV4社区版进行魔改  
源码太大坨了懒得上传，直接传成品好了。  
若是担心安全问题大可不必使用。

## 核心改动
1. 存储策略自定义
   - 支持用户前端直接切换存储策略
   - 支持指定文件夹设置单独策略
2. 分享权限增强
   - 文件夹分享支持设置匿名用户权限
3. 存储拓展
   - 存储策略支持对接123网盘（基于官方openapi）
